NLP pipeline

Raw text→ Lowercase + strip punctuation→ Remove stopwords→ Lemmatize→ Synonym expansion→ TF-IDF (1–2 grams, 6 000 features)→ Logistic Regression

Step-by-step

  1. 1
    Text cleaning — convert to lowercase, remove all punctuation via string.punctuation.
  2. 2
    Stopword removal — NLTK's English stopword list filters uninformative tokens (the, is, at…).
  3. 3
    Lemmatization — WordNetLemmatizer reduces inflected words to their base form (running → run).
  4. 4
    Synonym expansion — up to 2 WordNet synonyms per token are appended, enriching sparse vocabulary.
  5. 5
    TF-IDF vectorization — unigrams and bigrams across a 6 000-feature vocabulary, fitted on training data only.
  6. 6
    Logistic Regression — max_iter=400, 80/20 stratified split; outputs class probabilities for all three sentiments.
  7. 7
    Sector detection — keyword frequency counts from expanded_sector_keywords.csv determine industry; returns "Unknown" if no match.
  8. 8
    Auto reply — rule-based: Positive → thank-you message, Negative → apology, Neutral → acknowledgment.

Output visualisations

  1. A
    Pie chart — sentiment distribution across all inputs from the current session.
  2. B
    Per-sector word clouds — generated from session inputs grouped by detected sector.
  3. C
    Combined word cloud — all text in advanced_nlp_dataset.csv merged, 800×400 px, white background.

Interactive session steps

  1. 1
    The script trains the model on startup and prints accuracy to the console.
  2. 2
    At the prompt, type any customer review text and press Enter.
  3. 3
    It prints detected sector, predicted sentiment, per-class probabilities, and an auto-reply.
  4. 4
    Type exit to end the session — visualisations are generated automatically afterwards.
